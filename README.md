# 🇬🇧 UK Migration: Study & Work Applications Dashboard (2010–2023)

## 📌 Project Overview
This project presents an interactive **Power BI dashboard** analyzing migration trends into the United Kingdom, focusing on **study and work visa applications** between 2010 and 2023.

The dashboard provides a comprehensive view of:
- Global application distribution  
- Trends over time  
- Visa categories  
- Sponsorship patterns  
- Top contributing nationalities  

It enables stakeholders to explore migration insights through intuitive and dynamic visualizations.

---

## 🎯 Objectives
- Analyze global migration patterns into the UK  
- Compare **study vs work visa applications**  
- Identify top applicant nationalities  
- Examine institutional sponsorship trends  
- Provide interactive filtering for deeper insights  

---

## 🛠️ Tools & Technologies
- **Power BI** – Dashboard development & visualization  
- **DAX** – Calculations & KPIs  
- **Data Cleaning & Transformation** – Preprocessing and structuring  

---

## 📊 Dashboard Features

### 🌍 Global Applications Map
- Displays application distribution worldwide  
- Highlights country-wise migration intensity  

### 📈 Time-Series Analysis
- Study vs Work applications over time  
- Identifies growth trends and fluctuations  

### 📊 Applications by Year
- Yearly breakdown with growth indicators  
- Detects increases and declines  

### 🏫 Institution-Based Analysis
- Applications categorized by institution type  
- Higher education dominates sponsorship  

### 🌐 Nationality Insights
- Top countries applying for UK visas  
- Comparison across visa types  

### 🧭 Regional Breakdown
- Applications segmented by global regions  
- Highlights dominant migration corridors  

---

## 🎛️ Interactive Features
Users can filter data by:
- Nationality  
- Visa Type (Study / Work / Extensions / Unknown)  
- Year  

These filters enable customized and dynamic analysis.

---

## 📈 Key Insights
- Significant growth in applications over the years  
- Study visas dominate overall application volume  
- High concentration of applicants from Asia  
- Strong role of higher education institutions in sponsorship  
- Seasonal and policy-driven fluctuations in migration trends
